There is more to be said about AMF, but the things shown here should help you get started

Q: How did this module get loaded?
A: In pageManager.js a starting module is set to load in (bottom of the file), in many regards this can be considered the start of your AMF app.

Q: Why aren't you using ID's, aren't they better with JS?
A: AMF can be used to load multiple identical modules simultaniously, using ID's would break this
Q: Why are you using querySelector all the time?
A: querySelector has been demonstrated to be the fastest and most efficient for retrieving a single element with a specific class.
